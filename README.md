
# Extension Installation Guide

## Steps to Install

1. **Download the ZIP**

   * Get the extension ZIP file and extract it to a folder on your computer.

2. **Open Extensions Page**

   * Go to your browserâ€™s extensions page:

     ```
     <browser-name>://extensions
     ```

     *(Replace `<browser-name>` with your browser, e.g., `chrome`, `edge`, `brave`)*

3. **Turn on Developer Mode**

   * Enable **Developer mode** (usually a toggle in the top-right corner).

4. **Load Unpacked Extension**

   * Click **Load unpacked**.
   * Locate and select the folder where you extracted the extension.

5. **Enjoy!**

   * The extension is now installed and ready to use.
